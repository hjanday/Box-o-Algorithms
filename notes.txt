GitHub new repository to IntelliJ project?

1. Make repository

2. Clone

3. Open with IntelliJ

4. File --> Project Structure --> Set an SDK

5. File --> Project Structure --> Project Compiler Output, set this to an absolute path to the directory then add \out at the end

6. Right click directory name, add new folder named "src" (or anything really), right click the new folder, Mark Directory As --> "Sources Root"
